# Electronic Piano Circuit

A simple electronic piano built using the 555 Timer IC.  
The circuit generates different musical notes using push buttons, resistors, capacitors, and a speaker.

## Features
- 8 musical notes
- 555 Timer IC based sound generation
- Breadboard implementation
- Low-cost analog electronics project

## Components Used
- NE555 Timer IC
- Resistors
- Capacitors
- Push Buttons
- 8Ω Speaker
- Breadboard
- Jumper wires
- 5V Power Supply

## Applications
- Electronics learning
- DIY synthesizers
- Sound frequency experiments
- Educational projects
